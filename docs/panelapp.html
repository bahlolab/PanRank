PanRank - PanelApp Australia

Jacob E. Munro, Mark F. Bennett, Melanie Bahlo

PanelApp is a crowdsourcing tool to allow gene panels to be shared, downloaded, viewed and evaluated by the Scientific Community.

This page contains PanRank candidate gene rankings based on gene panels from PanelApp Australia. PanRank scores are between 0 and 1 and represent the predicted probability that a gene is disease-causing with the specified mode of inheritance. The “Known Inheritance” column indicates the known mode of inheritance used to train the model.
